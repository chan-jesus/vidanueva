vidanueva by chan-jesus

Church website in c++

Dependencies

wt-3.1.8 mongodb

Install

mkdir build cd build cmake .. ccmake .. # Do any manual config needed make -j4 /vidanueva --docroot="static-content" --no-compression --http-address=0.0.0.0 --http-port=8000 -c wt-config.xml # Go in your browser to http://localhost:8000
